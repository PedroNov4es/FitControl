# Sistema de Gerenciamento de Academia

Projeto desenvolvido em Java utilizando PostgreSQL.

## Tecnologias

- Java
- PostgreSQL
- JDBC
- Maven

## Funcionalidades

- Cadastro de alunos
- Cadastro de planos
- Cadastro de aulas
- Controle de frequência
- Validação de planos
- Relatórios
- Controle de inscrições

## Requisitos POO

- Herança
- Polimorfismo
- Encapsulamento
- Interface
- Classe abstrata
- Exceptions personalizadas
- Javadocs
